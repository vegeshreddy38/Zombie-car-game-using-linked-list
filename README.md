# ZOMBIE HIGHWAY - LINKED LIST VERSION
A browser-based game that demonstrates linked list data structures through gameplay.

# LIVE DEMO
https://mellow-dasik-60413f.netlify.app/

# GAME DESCRIPTION
Navigate your character through a highway filled with zombies. Use linked lists to manage the positions of the player and enemies on the game board.

# FILES
index.html : Main HTML structure and game layout
script.js : Game logic implementing linked list data structure
style.css : Styling and visual design
HOW TO PLAY
Open index.html in a web browser
Press Enter to start the game (auto-starts)
Use arrow keys (↑/↓) to move up and down between lanes
Press Space to shoot zombies
Avoid zombies to maintain health
Survive and score as high as possible
CONTROLS
↑ (Up Arrow) : Move up one lane
↓ (Down Arrow) : Move down one lane
Space : Shoot
Enter : Start game
Restart Button : Restart the game
# GAME SETTINGS
Lanes: 3
Columns: 10
Starting Health: 8
Spawn Chance: 40%
Tick Speed: 700ms
DATA STRUCTURE CONCEPT
This game uses linked lists to manage entities on the game board. Each lane contains a linked list where nodes represent either the player or zombies, demonstrating:

Node creation and deletion
List traversal
Dynamic insertion and removal
Memory-efficient entity management
# CREATED FOR
Data Structure learning and demonstration purposes.
